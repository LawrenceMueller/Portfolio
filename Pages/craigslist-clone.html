CraigslistClone

Web Application

A web application that scrapes results from a Craigslist search and uses that information in order to fill out its own webpage which is more aesthetically pleasing than Craigslist.

In order to scrape Craigslist, I used a third party library called Beautiful Soup which returns the HTML of a webpage in an easy to use object. This allowed me to easily parse the HTML based on HTML tags and CSS classes. The UI is just something I threw together using Materialize. The focus of the project was learning the basics of django which I believe I was successful in.

The project is built using the django and is my first project using django. I have seen that many companies are looking for django experience and I decided to start learning.
