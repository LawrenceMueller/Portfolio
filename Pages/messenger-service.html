MessengerService

Web Application

A full stack web service that sends users messages every day via email. The project was built using the MERN stack and is complete with payment processing via Stripe.

I utilize the Mongo database to store and access already stored user information. The Node REST webserver schedules emails to be sent, and handles user information sent to it by the front end, mainly validating it.

I planned, developed, produced, and am now maintaining this project. I am very proud of it. Feel free to check out the site itself by using the link below, or watch the video to see it in action. The video is from an earlier version of the website that sent messages via SMS, however the concepts are the same.
